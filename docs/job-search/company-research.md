# 🏢 Company-Specific Preparation: Hitting the Bullseye

Not all companies interview the same way.
Each category tests a **different skill signal**.

> Strategy mismatch = rejection
> Strategy alignment = selection

---

# 📊 The Big Three Company Types

---

## 🏢 1. Service-Based Giants

*(TCS, Infosys, Wipro, Accenture)*

### 🎯 What they test:

* Aptitude speed
* Basic coding ability
* Communication clarity

### 📉 DSA Level:

Basic (Arrays, Strings, Linked List)

### 🧠 Strategy:

* Previous Year Questions (PYQs)
* Speed > depth
* Accuracy in aptitude

📘 Practice sources:

* [GeeksforGeeks Company Interview Corner](https://www.geeksforgeeks.org/company-interview-corner/?utm_source=chatgpt.com)
* [PrepInsta TCS/Infosys Prep](https://prepinsta.com?utm_source=chatgpt.com)

---

## 💻 2. Product-Based Companies

*(Amazon, Google, Microsoft, Adobe)*

### 🎯 What they test:

* Deep DSA thinking
* Problem-solving under constraints
* System design (sometimes)

### 📈 DSA Level:

High (DP, Graphs, Trees)

### 🧠 Strategy:

* LeetCode-focused prep
* One strong programming language
* Explain trade-offs clearly

📘 Core practice sources:

* [LeetCode Interview Practice](https://leetcode.com/explore/interview/?utm_source=chatgpt.com)
* [NeetCode Roadmap](https://neetcode.io/roadmap?utm_source=chatgpt.com)
* [Blind 75 Problems List](https://leetcode.com/list/xoqag3yj/?utm_source=chatgpt.com)

---

## 🚀 3. High-Growth Startups

*(Zomato, Swiggy, Fintechs, SaaS)*

### 🎯 What they test:

* Real-world development ability
* Project depth
* Ownership mindset

### 📊 DSA Level:

Medium (practical problem-solving)

### 🧠 Strategy:

* Strong GitHub portfolio
* Project architecture clarity
* Take-home assignments performance

📘 Learning sources:

* [roadmap.sh Developer Roadmaps](https://roadmap.sh?utm_source=chatgpt.com)
* [System Design Primer](https://github.com/donnemartin/system-design-primer?utm_source=chatgpt.com)
