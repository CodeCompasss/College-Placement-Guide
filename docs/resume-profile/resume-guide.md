
# Resume Blueprint: From Zero to Hired

Your resume is not a document.

It is a **highly optimized interview filtering tool**.

> Recruiters spend ~6–10 seconds deciding whether to shortlist you.

---

# 🏗️ Structural Breakdown

---

## 1. 🧾 Header (Identity Layer)

Include only:

* Full Name (bold, large)
* Professional Email
* Phone Number
* LinkedIn
* GitHub
* Portfolio (optional)

❌ Avoid:

* Photos
* Personal details
* Overloaded contact info

---

## 2. 🎓 Education

Reverse chronological order:

* College Name
* Degree + Branch
* Graduation Year
* CGPA (only if strong)

Add relevant coursework:

* Data Structures
* DBMS
* Operating Systems
* Computer Networks

---

## 3. 🧠 Skills (ATS Keyword Layer)

Structured grouping:

* Languages: C++, Java, Python
* Frameworks: React, Node.js
* Tools: Git, Docker, AWS
* Database: MySQL, MongoDB

> Keep only job-relevant skills.

---

## 4. 💼 Experience (Impact Layer)

Use:

> Action + Task + Measurable Impact

### Example:

* Improved backend performance by 22% by optimizing queries and implementing Redis caching

---

## 5. 🚀 Projects (Most Important Section)

Include:

* 2–3 strong projects
* Tech stack
* Clear functionality
* Real impact
* GitHub link

---

# 📏 ATS Optimization Rules

* Standard headings only:

  * Experience
  * Education
  * Skills
  * Projects

❌ Avoid:

* Tables
* Icons overload
* Complex layouts
* Creative section names

---

# 🔥 Action Verbs

* Built
* Designed
* Engineered
* Developed
* Optimized
* Scaled
* Reduced
* Implemented
