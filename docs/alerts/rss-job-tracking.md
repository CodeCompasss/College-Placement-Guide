# RSS Job Tracking System

This is a **pro-level automation method** to track jobs without manually checking websites.

---

# 1. What is RSS Tracking?

RSS feeds automatically update you when:
- new jobs are posted
- companies update careers pages
- blogs announce hiring

---

# 2. Tools You Need

- Feedly (recommended)
- Inoreader
- Notion (optional integration)

---

# 3. Job RSS Sources

Add these sources:

## Job Boards
- Indeed RSS
- Glassdoor RSS
- RemoteOK RSS
- WeWorkRemotely RSS

## Company Pages
- Amazon Careers RSS (via tracking tools)
- Google Jobs updates
- Microsoft Careers feed

---

# 4. Setup Steps (Feedly Example)

1. Go to Feedly
2. Click “Add Content”
3. Paste job board URLs
4. Organize into folders:
   - Internships
   - SDE Jobs
   - Remote Jobs

---

# 5. Filtering System

Create rules:
- Title contains “Intern”
- Title contains “Engineer”
- Filter by “India” or “Remote”

---

# 6. Daily Workflow

- Open Feedly once daily
- Save relevant jobs
- Apply immediately

---

# 7. Advanced Automation (Optional)

Connect RSS → Notion:
- Auto-save job posts
- Track applications

---

# 8. Pro Tip

RSS is powerful because:
> You see jobs BEFORE they appear on LinkedIn feeds
